# Accenture AI Challenge – Self-Directed Practice Case

> Educational case study inspired by the **Data Scientist | Machine Learning & IA** position at [Accenture Brazil](https://www.linkedin.com/jobs/view/4231603926/). End-to-end ML, NLP, and cloud-scale pipeline simulation.  
> This project demonstrates an enterprise-scale **AI & Machine Learning pipeline** simulating Accenture's real-world Data & AI challenges — from data ingestion to MLOps and NLP with embeddings.
